# ✨ 我的星系之旅 · 使用说明

一个以「银河」为主题的个人主页，把你的分享化作星球，在宇宙中旋转发光。

---

## 🚀 快速开始

### 在浏览器中打开

直接用浏览器打开项目根目录下的 `index.html` 即可看到主页。

```
index.html      ← 个人主页
edit.html       ← 编辑页面（配置你的星系）
```

推荐使用 **Chrome / Edge / Safari** 等现代浏览器。

---

## 🪐 主页功能一览

### 1. 英雄区（Hero）

- 你的头像 + 专属星际身份卡（SPACE-XXXX）
- 点击身份卡右侧「复制」按钮可复制你的 ID，方便分享给好友
- 已连接的好友会在这里以发光行星的形式环绕你的头像

### 2. 银河碎片（我的星球）

- 你上传的每个「星球」，都会以行星的形式在轨道上旋转
- 用鼠标/手指 **点击某颗行星**，会打开该星球的图片画廊
- 在图片画廊里，**点击每张图片** 可以放大查看

### 3. 银河穿越区

纯视觉的装饰动画，模拟穿过银河的感受 🌀

### 4. 社交链接

- Instagram / Rednote / Douyin
- 可在编辑页自定义链接地址

### 5. 星际坐标探测仪（搜索好友）

- 在底部输入好友的 **星际坐标 ID**（格式：`SPACE-XXXX`）
- 点击「🔍 探测」按钮，找到好友后会弹出 TA 的主页
- 好友主页会展示 TA **公开的那颗星球**，点击可查看图片
- 你可以点击「✦ 连接」与好友建立连接，会有行星碰撞动画
- 连接成功后，TA 会出现在你的主页 hero 区，展示 TA **所有星球**
- 已连接用户可以点击「✧ 终止连接」取消连接

## 🛠 编辑你的星系

打开 `edit.html` 进行配置。

### 可编辑内容

| 项目 | 说明 |
|------|------|
| **头像** | 点击「更换头像」上传本地图片 |
| **个人坐标** | 自定义 ID 后缀（如 134340 则为 SPACE-134340） |
| **社交链接** | Instagram / Rednote / Douyin 的完整 URL |
| **星球** | 为每颗星球设置名字 + 上传多张图片 |
| **公开星球** | 选择哪一颗星球作为「公开星」（未连接的好友只能看到这颗） |

### 编辑流程

1. 打开 `edit.html`
2. 修改头像 / ID / 社交链接
3. 点击「+ 添加星球」新增星球卡片
4. 在每张星球卡片里填写星球名，并上传对应的图片
5. 勾选一张图片作为「公开星」（未连接好友仅可见这颗）
6. 点击底部「保存设置」上传到云端
7. 保存成功后会出现「分享链接」，复制发给好友即可

## 🔗 与好友建立连接

### 1. 分享你的坐标

在编辑页保存后，复制你的星际坐标 ID（如 `SPACE-134340`），发给好友。

### 2. 探测好友

在主页底部的 **星际坐标探测仪** 输入好友的 ID → 点击探测。

### 3. 查看好友主页

探测成功后会弹出好友的主页。

- **未连接时**：你只能看到好友的「公开星球」
- **连接后**：你可以看到好友的 **所有星球**，每颗都能点击查看图片

### 4. 发起连接

点击「✦ 连接」→ 两颗行星会发生碰撞动画 → 连接成功后：
- 好友主页关闭
- 回到你的主页
- Hero 区会多出一颗代表该好友的发光行星

### 5. 点击好友行星

点击 hero 区的好友行星，可重新打开 TA 的主页，查看 TA 的所有星球图片。

### 6. 终止连接

在好友主页中点击「✧ 终止连接」即可取消连接。

## 🔌 数据存储说明

本项目使用 **Supabase** 作为云端数据存储（数据库 + 文件存储）。

- 用户资料（头像、ID、社交链接）保存在数据库
- 星球信息（名称、图片）和图片文件保存在云端
- 好友连接关系也保存在数据库中

### 如果你想搭建自己的版本

1. 在 [supabase.com](https://supabase.com) 注册账号，创建项目
2. 在项目中创建一个数据库表 `profiles` 和 `planets`、`connections`
3. 创建一个存储桶 `avatars` 和 `planet-images` 用于存放图片
4. 修改 `supabase-config.js` 中的 `SUPABASE_URL` 和 `SUPABASE_ANON_KEY` 为你自己的值
5. 在 `supabase-connections.sql` 中运行 SQL 脚本，建立用户连接表和安全策略

> 详细的数据库配置可以参考项目内的 `SUPABASE_SETUP.md`。

## 💡 使用小技巧

1. **公开星球建议选你最想展示的一张**：未连接的朋友会以这颗星球认识你
2. **ID 尽量简短好记**：方便好友搜索
3. **首次使用**：先去 `edit.html` 编辑并保存一次，你的主页才会在搜索中被他人找到
4. **重复连接会防抖**：系统不会重复响应过快的点击
5. **图片数量**：每颗星球可以上传多张，建议不超过 20 张以保证加载速度

## ❓ 常见问题

**Q: 我点了连接但画面没反应？**
A: 可能网络请求较慢，但最新版本已优化为不阻塞动画流程，稍等几秒即可。如仍无效果，刷新页面重试。

**Q: 搜索好友找不到？**
A: 请确认好友已经在 `edit.html` 中「保存设置」过（只有保存后才会被搜索到），且 ID 输入正确。

**Q: 点击行星看不到图片？**
A: 该星球可能还没有上传图片，或只有公开星球可见（未连接状态）。

**Q: 怎么删除某个星球？**
A: 在编辑页中，点击星球卡片右上角的删除按钮即可。
